## Технические характеристики

### Производительность:
- **Время инференса**: 0.0068с (147 FPS)
- **Полный конвейер**: 0.0169с (59 FPS)
- **Устройство**: Apple Silicon (MPS)

### Модель:
- **Архитектура**: ResNet18 (pretrained)
- **Классы**: 7 эмоций
- **Оптимизация**: AdamW + CosineAnnealingLR
- **Transfer Learning**: Fine-tuning layer4

### Метрики качества:
- Accuracy
- F1-score (macro)
- Confusion Matrix
- ROC AUC (по классам)
- Classification Report
